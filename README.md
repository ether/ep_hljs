# ep_syntax_highlighting

Whole-pad syntax highlighting for Etherpad. Pick a language or let highlight.js auto-detect; the choice syncs across all collaborators on the pad. HTML/PDF exports include the highlighting.

## Install

```bash
pnpm run plugins i ep_syntax_highlighting
```

Performance numbers on 100 / 500 / 1000 / 5000 line pads land in this README before v1.0.0.
